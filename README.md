#How to Fork

+ Clone this repository.
+ Add file `src/twitter4j.properties` and write your application's API keys.

```
# twitter4j.properties

oauth.consumerKey=YOUR_API_KEY
oauth.consumerSecret=YOUR_API_KEY_SECRET
```
